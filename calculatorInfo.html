Panduan Kalkulator Limit Fungsi

Panduan Penggunaan Kalkulator Limit Fungsi & Grafik Interaktif

1. Masukkan Fungsi (f(x))

Pada bagian "Function (f(x))", masukkan fungsi matematika yang ingin Anda hitung limitnya.

Contoh input: sin(x)/x, x^2, log(x).

2. Masukkan Nilai Pendekatan x

Pada bagian "Limit as x approaches", masukkan nilai pendekatan x yang ingin Anda gunakan untuk menghitung limit.

Contoh: Jika Anda ingin menghitung lim (x → 0) sin(x)/x, maka masukkan 0 di kolom tersebut.

Input lain: 1, 2, ∞ (untuk mendekati tak hingga).

3. Gunakan Tombol Simbol Matematika (Opsional)

Jika ingin memasukkan simbol matematika umum seperti x, √, sin, log, Anda dapat menggunakan tombol-tombol yang tersedia di kalkulator.

Klik tombol simbol untuk otomatis menambahkannya ke input fungsi.

4. Menghitung Limit

Setelah memasukkan fungsi dan nilai pendekatan, klik tombol "Calculate Limit" untuk menghitung limit dari fungsi yang telah dimasukkan.

Hasil perhitungan limit akan muncul di bawah kalkulator, dalam format:

Limit as x → [nilai yang dimasukkan]: [hasil]

Contoh: Limit as x → 0: 1 untuk fungsi sin(x)/x.

5. Menampilkan Grafik Fungsi

Selain menghitung limit, kalkulator ini juga akan menampilkan grafik fungsi yang telah dimasukkan.

Grafik akan ditampilkan pada bagian bawah dengan label sumbu x dan f(x).

Anda bisa melakukan zoom in dan zoom out pada grafik untuk melihat lebih detail perilaku fungsi.

Contoh Kasus Penggunaan:

1. Limit dari Fungsi Trigonometri

Fungsi: sin(x)/x

Nilai pendekatan x: 0

Masukkan fungsi di kolom input dan angka 0 di kolom pendekatan. Tekan "Calculate Limit".

Hasil: Limit as x → 0: 1 dan grafik akan muncul menunjukkan perilaku fungsi di sekitar x = 0.

2. Limit dari Fungsi Logaritma

Fungsi: log(x)

Nilai pendekatan x: 1

Masukkan fungsi di kolom input dan angka 1 di kolom pendekatan. Tekan "Calculate Limit".

Hasil: Limit as x → 1: 0, beserta grafik yang menunjukkan perilaku fungsi di sekitar x = 1.

Tombol Simbol Matematika:

  • x: Variabel umum untuk fungsi.
  • ^: Untuk pangkat atau eksponen. Misalnya, x^2 untuk x².
  • √: Akar kuadrat. Misalnya, sqrt(x) untuk √x.
  • sin: Fungsi sinus.
  • cos: Fungsi kosinus.
  • log: Fungsi logaritma.

Kesalahan Input:

Jika terdapat kesalahan input seperti fungsi tidak valid atau penulisan yang salah, kalkulator akan menampilkan pesan error. Pastikan untuk memeriksa kembali format fungsi yang dimasukkan.

Pastikan Anda menggunakan tanda kurung "( )"
